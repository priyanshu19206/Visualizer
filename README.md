Bubble Sort Visualizer -
An interactive web-based visualizer to understand how the Bubble Sort algorithm works through animations.

Features -
* Visual representation of array using bars
* Real-time Bubble Sort animation
* Color indication for comparison and swapping

How Bubble Sort Works - 
O(n^2)
Repeatedly compares adjacent elements
Swaps them if they are in the wrong order
Largest elements “bubble up” to the end

Tech Stack
* HTML
* CSS
* JavaScript
